🎯 Kurze Pumpgun-Actions
Schnelle Ein-Buchstaben-Befehle für tägliche Aufgaben:

c → Terminal leeren
q → Terminal beenden
h → Befehlshistorie anzeigen
x → Skript ausführbar machen
reboot → System sofort neu starten
🏠 Navigation & Verzeichnisse
Schnelle Wechsel zu wichtigen Verzeichnissen:

alex, home → Wechsel zu /home/alex/
vera → Wechsel zu VeraCrypt-Mountpoint
dvd, cd2 → Eine Ebene höher + Auflistung
var, log, logs → Wechsel zu Systemlogs
etc → Wechsel zu Systemkonfiguration
📦 Paket-Management
Effizientes Software-Management:

install-cmmn → Installiert häufig benötigte Tools
install-spcl → Installiert spezielle Tools
install, uninstall → Paket-Installation/Deinstallation
update, upgrade → Systemupdates
💻 System-Monitoring
Performance-Überwachung auf einen Blick:

systemstatus → Top-Prozesse anzeigen
ram → RAM-Übersicht
platz, platte → Festplattenplatz
top, btop, htop → Btop-Prozessmonitor
psaux, psax → Alle Prozesse anzeigen
🛠 Prozess-Management
Prozesskontrolle und Fehlerbehebung:

killforce, exorzist → Prozess sofort beenden
bitte, pls → Letzten Befehl mit sudo wiederholen
hg, history-grep → Historie durchsuchen
🎨 Verschönerung & Sicherheit
Farbe und Schutz für deine Terminal-Erfahrung:

grep, egrep, fgrep → Farbliche Hervorhebung
ls, l, sl → Farbige Auflistung
cp, mv, rm → Interaktiver Modus (Schutz vor Überschreiben)
